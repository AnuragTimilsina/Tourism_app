<!-- GETTING STARTED -->
## Getting Started


### Prerequisites

Installation: 
* pip
  ```sh
  pip install -r requirements.txt
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/AnuragTimilsina/Tourism_app.git
   ```
3. Install PIP packages
   ```sh
   pip install -r requirements.txt
   ```
4. Run this django project `manage.py`
   ```sh
   python manage.py runserver
   ```
